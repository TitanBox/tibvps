Language

Bash shell

Description

This script is used base on VestaCP or without it. For automatically installing, checking running status, uninstalling ... web dev softwares like memcached, mysql, postgresql, apache ... without putting a hand on any command lines.

Installation

curl -O https://raw.githubusercontent.com/TitanBox/tibvps/master/latest/tib

Example

Check existence 

// check if vesta installed or check if the software supports your pc
bash tib vesta
--> service:vesta is not installed or action: is not allowed.
bash tib vesta supported
--> supported

Install a software 

// install VestaCP
bash tib vesta install
// install Memcached
bash tib memcached install

Get information 

// get postgresql real service name or check if it is running
bash tib postgre name
--> postgresql
bash tib postgre running
--> running

Uninstall software 

// fresh uninstall for vestaCP
bash tib vesta uninstall

Ultilites 

// os name
bash tib os name
// os version
bash tib os version
// reset all ip config
bash tib ip reset
// save current ip config
bash tib ip save

And many other things. Use it with your freedom and wisdom

Update

This script will update to the latest version when you call

bash tib update

By the way, if you want to see software credit

bash tib about

Good luck
